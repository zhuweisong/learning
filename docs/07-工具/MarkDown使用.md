# markdown的使用

## 信息源 (官方网站) {#custom-id}

[英文站](https://www.markdownguide.org/)
[中文站](https://markdown.com.cn/intro.html)

## 优势

- GitHub等各主流厂支持
- 纯文本可移植
- 语法简单
- 专注于文字，便于git存放

## 劣势

- 没有像腾讯文档、PPT这种起步方便;
- 对非领域内人员，非所见所得稍麻烦

## 实现原理

- 使用MarkDown解析器，将文件转换为HTML格式
- 使用浏览器进行显示

## VSCode MD扩展

- Markdown All in One MD: 增强多合一插件.
- Markdown Preview Enhanced: 预览增强
- Markdown Image: MD 快捷粘贴引用本地图片.
- markdownlint: 语法检查
